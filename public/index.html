TinyShade examples

Explore a curated collection of real-time GPU experiments built with TinyShade. Each example opens a standalone HTML file showcasing a specific rendering technique, algorithm, or engine feature—from procedural noise and fractals to raymarching, path tracing, and audio-visual synthesis.

Example 1

This example demonstrates Domain Warping combined with Voronoi (Cellular) Noise to create a visceral, organic "Bio-Organic Membrane" effect. It showcases how simple mathematical functions can be layered to simulate complex, natural textures.

Example 2

This example showcases a Fractal Orbit Trap rendered with high-precision Anti-Aliasing and Distance Estimation. While standard fractals like the Mandelbrot set typically use "escape time" (counting how long it takes to fly to infinity), this shader uses the path of the points (the "orbit") to create intricate, textured coloring.

This example builds upon the organic membrane simulation by introducing Temporal Accumulation and a Post-Process Bloom Pass. It demonstrates how to transform sharp, mathematical noise into a soft, cinematic "bioluminescent" environment by leveraging the history of previous frames and simulated lens optics.

Example 5

This system achieves a "Warp Drive" aesthetic by syncing a 1-million particle compute simulation with fractal geometry. Since both passes share the same Signed Distance Function (SDF), particles "sense" the fractal; they calculate their distance to the Mandelbox and reset upon impact, preventing clipping and creating a realistic collision effect.
